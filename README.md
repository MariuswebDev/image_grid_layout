# Image Grid Layout

https://roadmap.sh/projects/image-grid
A simple responsive image gallery built with **HTML** and **CSS Grid**. This project demonstrates how to create a modern masonry-style image layout using CSS Grid, with images of varying heights arranged in an aesthetically pleasing gallery.

## Features

- Responsive CSS Grid layout
- Multiple image sizes and aspect ratios
- Clean and modern design
- Lightweight and easy to customize
- Pure HTML and CSS (no JavaScript required)

## Technologies Used

- HTML5
- CSS3
  - CSS Grid
  - Responsive Design
  - Object Fit

## Project Structure

```text
.
├── index.html
├── style.css
└── images/
```

## Customization

You can easily:

- Add more images to the gallery
- Adjust grid columns and spacing
- Modify image sizes using grid row and column spans
- Customize colors, borders, and hover effects

## Learning Objectives

This project was created to practice:

- CSS Grid Layout
- Responsive Web Design
- Image Positioning
- Modern UI Development
